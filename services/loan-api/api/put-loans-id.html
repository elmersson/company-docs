PUT /loans/:id New

Updates an existing loan by ID.

Authentication & Validation

  • Authentication: Required (auth middleware)
  • Validation: Request body validation enabled

Path Parameters

Parameter Type Required Description
id string Yes The unique identifier of the loan to update

Example Request

PUT /loans/12345 HTTP/1.1
Authorization: Bearer your-jwt-token
Content-Type: application/json

{
  "amount": 15000,
  "interestRate": 5.5,
  "term": 24,
  "status": "approved"
}

Example Response

HTTP/1.1 200 OK
Content-Type: application/json

{
  "id": "12345",
  "amount": 15000,
  "interestRate": 5.5,
  "term": 24,
  "status": "approved",
  "updatedAt": "2024-01-15T10:30:00Z"
}
