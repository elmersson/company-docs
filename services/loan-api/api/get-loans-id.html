GET /loans/:id New

Description

Retrieve a specific loan by ID.

Path Parameters

Parameter Type Required Description
id string Yes The unique identifier of the loan

Example Request

GET /loans/12345

Example Response

{
  "id": "12345",
  "amount": 50000,
  "status": "active",
  "createdAt": "2024-01-15T10:30:00Z"
}
