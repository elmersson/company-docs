GET /loans/:loanId/payments/:paymentId New

Retrieves details for a specific payment within a loan.

Path Parameters

Parameter Type Required Description
loanId string Yes Unique identifier for the loan
paymentId string Yes Unique identifier for the payment

Example Request

GET /loans/loan-123/payments/payment-456
Host: api.example.com
Accept: application/json

Example Response

{
  "id": "payment-456",
  "loanId": "loan-123",
  "amount": 1250.00,
  "paymentDate": "2024-01-15",
  "status": "completed",
  "paymentMethod": "bank_transfer"
}
