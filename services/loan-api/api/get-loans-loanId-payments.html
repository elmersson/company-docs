GET /loans/:loanId/payments New

Retrieves all payment records for a specific loan.

Path Parameters

Parameter Type Description
loanId string The unique identifier of the loan

Response

Returns an array of payment records associated with the specified loan.

Example Response

[

    "id": "payment-123",
    "loanId": "loan-456",
    "amount": 1250.00,
    "date": "2024-01-15T10:30:00Z",

    "method": "bank_transfer"
  },
  {
    "id": "payment-124",
    "loanId": "loan-456",
    "amount": 1250.00,
    "date": "2024-02-15T10:30:00Z",
    "status": "pending",
