PaymentDto New

Data transfer object representing a payment transaction in the loan system.

Field Type Required Description
id string Yes Unique identifier for the payment transaction
loanId string Yes Reference to the associated loan
amount number Yes Payment amount in the base currency
method PaymentMethod Yes Payment method used for the transaction
status PaymentStatus Yes Current status of the payment
processedAt string Yes Timestamp when the payment was processed
reference string | undefined No Optional external reference number
