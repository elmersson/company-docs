LoanDetailDto New

Data transfer object containing comprehensive loan information and applicant details.

Field Type Required Description
id string Yes Unique identifier for the loan
amount number Yes Principal loan amount requested
termMonths number Yes Loan term duration in months
interestRate number Yes Annual interest rate as decimal
status LoanStatus Yes Current status of the loan application
applicantName string Yes Full name of the loan applicant
applicantEmail string Yes Email address of the loan applicant
employmentStatus EmploymentStatus Yes Current employment status of applicant
createdAt string Yes Timestamp when loan was created
updatedAt string Yes Timestamp of last loan update
