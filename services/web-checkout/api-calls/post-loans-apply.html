POST /loans/apply New

Consumes: loan-api

Submits a new loan application from the checkout flow. Called when the user completes the loan application form and clicks "Apply".

Request

Sends a LoanApplicationDto as the request body.

FieldTypeDescription
amountnumberRequested loan amount
purposestringPurpose of the loan
termnumberLoan term in months

Response

Returns a LoanResultDto with the application result.

Called from: src/api/loan.api.ts
